# KDTree (k-dimensional tree)

#### 개요
KDTree와 google API를 이용하여 내 위치에서 가까운 거리순으로 주변 위치정보를 가져온다.  

#### 상세설명
https://en.wikipedia.org/wiki/K-d_tree

#### 사용언어
Python 3.7

#### 실행방법
```
python go.py
```

#### Dependency

```
pip install requests==2.21.0  
pip install scipy==1.1.0  
pip install numpy==1.16.4  
```
